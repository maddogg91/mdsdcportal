Websites

Packages start at $150+ tax
Static/Dynamic Web served pages



Static websites can be delivered fairly quickly and do not require a backend sever. Dynamic websites such as sites needing web forms, search forms, etc require a server and will not work on some web hosting sites.

Web Applications

Packages start at $350+ tax
Complex web served applications



Web apps are more complex web served applications. Popular examples include login/registration portals requiring database connectivity and live data management or web pages requiring Google or Facebook API's.

Mobile Applications

Packages start at $500+ tax
Simple mobile applications



Simple Android or IOS apps, due to the scope of many applications, mobile apps can take the longest for delivery due to testing on multiple platforms.
